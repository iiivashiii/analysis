# Анализ финансовой эффективности компаний по ОКВЭД и ИНН

Этот проект представляет собой автоматизированный аналитический конвейер, написанный на Python. Программа позволяет находить организации по коду деятельности, массово собирать их финансовые отчеты по ИНН, рассчитывать бизнес-показатели и визуализировать результаты на интерактивной карте.

## 🔍 Описание работы программы

Процесс анализа разделен на несколько ключевых этапов:

1. **Сбор базы компаний**: Скрипт выполняет поиск в ГИР БФО по заданному ОКВЭД (например, 41.20 — строительство) и формирует список ИНН и уникальных идентификаторов организаций.
2. **Парсинг финансовых данных по ИНН**: 
   * Для каждого найденного ИНН программа обращается к API налоговой службы.
   * Собираются данные из "Отчета о финансовых результатах": выручка (2110), себестоимость (2120), налоги (2410) и прочие показатели.
   * На основе собранных данных автоматически рассчитывается показатель **EBITDA**.
3. **Аналитика**: Скрипт сравнивает показатели компаний за разные периоды (2021 и 2024 годы), вычисляет темпы роста и рентабельность (EBITDA Margin).
4. **Визуализация**:
   * Построение гистограмм распределения роста в отрасли.
   * Создание интерактивной карты России с использованием **Folium**.
   * Каждая компания на карте представлена маркером, размер которого зависит от выручки, а цвет указывает на эффективность бизнеса относительно медианы рынка.

## 🛠 Стек технологий

* **Язык**: Python 3.12[cite: 2]
* **Библиотеки**: Pandas (анализ данных), Plotly (графики), Folium (карты), Requests (API запросы), ThreadPoolExecutor (многопоточность), Dotenv (безопасность).
* **Геокодирование**: DaData API для получения координат штаб-квартир по ИНН.
